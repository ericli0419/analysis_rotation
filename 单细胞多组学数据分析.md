# 单细胞多组学数据分析

## 分析环境

### Python

```shell
# SCANPY
# https://scanpy.readthedocs.io/en/latest/index.html
pip install scanpy

# cellrank
# https://cellrank.readthedocs.io/en/stable/index.html
pip install cellrank
# or with extra libraries, useful for large datasets
pip install 'cellrank[krylov]'

# scvelo
# https://scvelo.readthedocs.io/
pip install scvelo

# multivelo
# https://multivelo.readthedocs.io/en/latest/
pip install multivelo
```
